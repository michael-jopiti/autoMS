# autoMS
LC-MS tool for fast, automated mzML preprocessing in untargeted metabolomics analysis.


## Install and launch
```bash
git clone https://github.com/michael-jopiti/autoMS.git
echo "alias autoMS='python3 $PWD/autoMS/main.py'" >> ~/.bashrc
source ~/.bashrc
autoMS -h
```
